## 整型

1. 整數類型:
   - Go 提供有符號和無符號的整數運算。
   - 有符號整數: int8, int16, int32, int64 (數字表示 bit 大小)。
   - 無符號整數: uint8, uint16, uint32, uint64。

1. 特定CPU平台整數類型:
   - int 和 uint 都可以是 32 或 64 bit，但確切大小依賴於編譯器和硬件。
   - 不應假設 int 和 uint 的具體大小。

2. 其他整數類型:
   - rune 是 int32 的等價類型，主要表示 Unicode 碼點。
   - byte 是 uint8 的等價類型，強調是原始數據而不是小整數。

3. 指針大小整數類型:
   - uintptr 的大小足以容納指針，但具體大小未指定。
   - 主要在底層編程或與 C 語言交互時使用。

4. 整數範圍:
   - 有符號整數使用2的補碼形式，例如：int8 的範圍是 -128 到 127。
   - 無符號整數全部 bit 表示非負數，例如：uint8 的範圍是 0 到 255。

5. 類型轉換:
   - 即使 int 和 int32 可能有相同大小，它們仍然是不同的類型。
   - 轉換 int 到 int32 或反向都需要明確的類型轉換。

下面是Go語言中關於算術運算、邏輯運算和比較運算的二元運算符，它們按照先級遞減的順序的排列：

```
*      /      %      <<       >>     &       &^
+      -      |      ^
==     !=     <      <=       >      >=
&&
||
```
1. 運算符優先級:

    - 有五種優先級。
    - 同一優先級使用左優先結合規則。
    - 括號可用於明確優先順序或提高優先級，如 mask & (1 << 28)。

2. 賦值與運算符結合:

    - 一些運算符（如 +）有與賦值結合的對應運算符 (+=)，用於簡化賦值語句。

3. 算術運算符:

    - +、-、* 和 / 可用於整數、浮點數和複數。
    - % 取模運算符僅用於整數運算。

4. %取模運算符:

    - Go 中的 % 的行爲與其他語言可能不同。
    - % 的符號與被取模數的符號總是一致的，例如 -5%3 和 -5%-3 結果都是 -2。

5. 除法運算符:

    - / 的行爲依賴於操作數的類型。
    - 若操作數均爲整數，例如 5/4，結果是 1。
    - 若操作數中有浮點數，例如 5.0/4.0，結果是 1.25。

6. 溢出:

    - 如果運算結果需要更多的bit位才能表示，則認爲該結果是溢出的。
    - 超出的高位bit位部分會被丟棄。


```
var u uint8 = 255
fmt.Println(u, u+1, u*u) // "255 0 1"

var i int8 = 127
fmt.Println(i, i+1, i*i) // "127 -128 1"

```
- 使用了uint8類型，這是一個8位無符號整數，其值域是0到255。
    - u 是 255，所以直接印出是 255。
    - u+1：當你加1到255（最大的uint8值），結果會回繞（wrap around）到0，因此印出是 0。
    - u*u：255 * 255 = 65025，但因為 uint8 只能表示0到255之間的數字，所以這會回繞到 65025 % 256 = 1，因此印出是 1。


- 使用了int8類型，這是一個8位有符號整數，其值域是-128到127。
  - i 是 127，所以直接印出是 127。
  - i+1：當你加1到127（最大的int8正值），結果會溢出並變成int8的最小值，即 -128。
  - i*i：127 * 127 = 16129，但因為 int8 只能表示-128到127之間的數字，所以這會回繞到 16129 % 256 - 128 = 1，因此印出是 1。


兩個相同的整數類型可以使用下面的二元比較運算符進行比較；比較表達式的結果是布爾類型。

```
==    equal to
!=    not equal to
<     less than
<=    less than or equal to
>     greater than
>=    greater than or equal to
```
對於整數，+x是0+x的簡寫，-x則是0-x的簡寫；對於浮點數和複數，+x就是x，-x則是x 的負數。

Go語言還提供了以下的bit位操作運算符，前面4個操作運算符併不區分是有符號還是無符號數：

```
&      位運算 AND
|      位運算 OR
^      位運算 XOR
&^     位清空 (AND NOT)
<<     左移
>>     右移
```
